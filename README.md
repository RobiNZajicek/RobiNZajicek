### About Me

Full-Stack Developer building **[Mubite](https://mubite.com)** — a multi-language cryptocurrency prop trading platform. I own the entire stack end-to-end: frontend, backend, database, DevOps, payments, and third-party integrations. Before Mubite, I spent 2+ years building websites and web apps for clients at Nara Webs.

---

### What I Built at Mubite

- **Trading challenge system** with multi-phase evaluation, profit calculators, and real-time phase progress tracking
- **Bybit API integration** — connecting user accounts, syncing trades, and monitoring positions via WebSockets
- **Payment processing** — PayPal, CoinPayments, Confirmo crypto gateway, discount codes, and affiliate payouts
- **Trader dashboard** — portfolio analytics, trade history, withdrawals, onboarding checklists, Bybit connection flow
- **Affiliate marketing system** — multi-level referral tracking with automated commission payouts
- **CMS & SEO** — blog/articles system, dynamic sitemap generation, structured data, Trustpilot reviews
- **i18n** — full internationalization with Next.js i18n routing and language switcher
- **Infrastructure** — Docker containers (Node.js + MariaDB + nginx), Redis caching, rate limiting, Brevo transactional emails, Firebase auth, Cloudflare Turnstile, S3 storage
